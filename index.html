About the Project

This simulator demonstrates the working principle of a Low Power ALU designed using Verilog HDL for VLSI applications. It allows users to perform basic arithmetic and logical operations, the same operations carried out by the actual RTL design, giving a hands-on feel for how an Arithmetic Logic Unit processes data at the hardware level.

Features

Arithmetic Operations

Performs addition and subtraction with accurate results.

Logical Operations

Supports AND, OR, XOR, and NOT operations.

Low Power Design

Based on an ALU architecture optimized for reduced power consumption.

Verilog RTL Design

Reflects the logic implemented in the original Verilog HDL design.
